# Spatial Category Learning

This repository contains the behavioral experiments, computational modeling, and neuroimaging analysis code developed as part of my PhD research investigating how spatial representations influence category learning.

The central question motivating this work is whether structured spatial experience can shape the internal representations that support later concept and category learning. The project combines behavioral experiments, connectionist modeling, and functional MRI to investigate this question from complementary perspectives.

## Research Components

### Behavioral

A series of behavioral experiments testing whether different spatial layouts influence subsequent category learning, including manipulations of spatial structure and stimulus-response congruency.

### Computational Modeling

Neural network models examining whether representational geometry acquired during spatial learning is sufficient to account for observed behavioral effects.

### fMRI

An ongoing neuroimaging investigation of how spatial learning shapes neural representations during later category learning using univariate and multivariate analysis techniques.
